Overview

Integral tertentu dan tak tertentu

Integral tertentu dan tak tertentu Integral:

  • tak tertentu: Jika \(f(x)\) ditentukan maka setiap fungsi \(F(x)\) hingga \(F'(x) = f(x)\) disebut Integral tak tertentu dari \(f(x)\) dituliskan \(\int f(x)= F(x) + C\).
    Contoh: \(x^3, x^3-2, x^3+1\) adalah integral tak tertentu dari \(f(x)=3x^2\), dituliskan \(\int 3x dx= x^3 + C\)

  • tertentu: Jika \(f(x)\) fungsi kontinyu pada \([a, b]\) maka dikatakan bahwa \(f(x)\) terintegralkan (integrable) pada \([a, b]\) dan disebut integral tertentu dan dinyatakan dengan \(\int \limits_a^{b}f(x) dx\).

Sifat-sifat integral tak tertentu

Sifat-sifat integral tak tertentu

  1. dimana \(k\) konstanta
    Bukti: Untuk sisi kiri, \(\dfrac {d \{ \int k f(x) dx \} } {dx} = k f(x)\)
    Untuk sisi kanan,
    \(\dfrac {d \{ k \int f(x) dx \} } {dx} = \dfrac {kd \{ \int f(x) dx \} } {dx}=k f(x)\)


  2. Bukti: Untuk sisi kiri, \(\dfrac {d \bigl[\int \{ f(x) \pm g(x) \} dx \bigr]} {dx} = f(x) \pm g(x)\)
    Untuk sisi kanan,
    \(\dfrac {d \bigl[\int f(x) dx \pm \int g(x) dx \bigr]} {dx} = \dfrac {d \int f(x) dx}{dx} \pm \dfrac {d \int g(x)}{dx}\)
    \(\hspace{118pt} = f(x) \pm g(x)\)

Beberapa rumus integral tak tertentu

Beberapa rumus integral tak tertentu[1]

  1. \(\int 0 dx= C\)

  2. \(\int x^n dx= \dfrac{1}{n+1} x^{n+1} +C;~~(n\neq -1)\)

  3. \(\int \dfrac{1}{x} dx = \ln |x| + C\)

  4. \(\int e^xdx = e^x + C\)

  5. \(\int a^x dx = \dfrac {a^x} {\ln x} + C\)

  6. \(\int \sin x dx = - \cos x + C\)

  7. \(\int \cos x dx = \sin x + C\)

  8. \(\int \tan x dx = \ln | \sec x | + C\)

  9. \(\int \cot x dx = \ln | \sin x | + C\)

  10. \(\int \sec x dx = \ln | \sec x + \tan x | + C\)

  11. \(\int \csc x dx = \ln | \csc x - \cot x| + C\)

Beberapa rumus integral tak tertentu[2]

  1. \(\int \sec ^2 x dx = \tan x +C\)

  2. \(\int \csc ^2 x dx = - \cot x + C\)

  3. \(\int \sinh x dx= \cosh x + C\)

  4. \(\int \cosh x dx= \sinh x + C\)

  5. \(\int \tanh x dx= \ln |\cosh x| + C\)

  6. \(\int \coth x dx= \ln |\sinh x| + C\)

  7. \(\int sech^2 x dx= \tanh x +C\)

  8. \(\int csch^2 x dx= - \coth x + C\)

  9. \(\int \dfrac{dx}{\sqrt{1-x^2}} = \arcsin x + C\)

  10. \(\int \dfrac{dx}{\sqrt{a^2-x^2}} = \arcsin \dfrac{x}{a} + C\)

  11. \(\int \dfrac{dx}{1+x^2} = \arctan x + C\)

Beberapa rumus integral tak tertentu[3]

  1. \(\int \dfrac{dx}{a^2+x^2} = \dfrac{1}{a} \arctan \dfrac{x}{a} + C\)

  2. \(\int \ln x dx = x \ln x - x + C\)

  3. \(\int \dfrac{dx}{\sqrt{x^2 \pm 1}} = \ln | x+ \sqrt{x^2 \pm 1} |+ C\)

  4. \(\int \dfrac{dx}{\sqrt{x^2 \pm a^2}} = \ln | x+ \sqrt{x^2 \pm a^2} |+ C\)

  5. \(\int \sqrt{1-x^2 } = \dfrac{x}{2}\sqrt{1-x^2 }+ \dfrac{1}{2} \arcsin x + C\)

  6. \(\int \sqrt{a^2-x^2 } = \dfrac{x}{2}\sqrt{a^2-x^2 }+ \dfrac{a^2}{2} \arcsin \dfrac {x}{a} + C\)

Integrasi parsial

Integrasi parsial Jika \(u=f(x)\) dan \(v=g(x)\) adalah fungsi-fungsi yang diferensiabel maka:

Cara menggunakan rumus diatas:

  • \(dv\) dipilih sehingga v mudah dicari

  • \(\int v du\) harus menjadi lebih mudah daripada \(\int u dv\)

Contoh:

  • \(I = \int x e^x dx\)

  • \(I = \int e^x \sin{x} dx\)

Penerapan \(1/D\) pada integral tak tertentu

Penerapan \(1/D\) pada integral tak tertentu \[D= \dfrac{d}{dx}~~operator~ derivatif\]
\(\dfrac{1}{D} = D^{-1} = \int ... dx\) operator integral
\(\dfrac{1}{1-D}=1 + D+ D^2+ ...; \Longrightarrow \dfrac{1}{1+D} = 1 - D - D^2-...\)
\(D \sin ax = a \cos ax\);
\(D \cos ax = -a \sin ax\);
\((D^2-b^2) \sin ax = (-a^2 - b^2) \sin ax\)
\((D^2-b^2) \cos ax = (-a^2 - b^2) \cos ax\), sehingga:

Rumus: Bukti : \[D(e^{ax} U) = ae^{ax} U + e^{ax} DU\] \[D(e^{ax} U) = e^ax(D + a) U\]

\[D\Bigl\{e^{ax}\dfrac{1}{(D+a)}V\Bigr\} = ea^{ax} V\]

Misal: \[(D+a) U = V \Longrightarrow U = \dfrac{1}{D+a} V\] \[\dfrac{1}{D}(e^{ax}V)= e^{ax}\dfrac{1}{(D+a)} V\]

Buktikan:
\(\int e^{ax}\cos{bx} dx=e^{ax}(D-a)\dfrac{\cos{bx}}{-b^2-a^2}\) \(\int e^{2x}(2x^2-4x+8)dx = \dfrac{1}{D}e^{2x}(2x^2-4x+8)\)

Rumus: Bukti:
\(D(x~U) = x ~DU + U\) Misal : \(DU = V \rightarrow U= \dfrac{1}{D}\)

\[\begin{aligned} D(x \dfrac{1}{D}xV) &= x V + \dfrac{1}{D}V \\ x\dfrac{1}{D}V &= \dfrac{1}{D} (x V) + \dfrac{1}{D^2}V\\ \dfrac{1}{D} (x V) &= x \dfrac{1}{D}V - \dfrac{1}{D^2} V \\ \dfrac{1}{D} (x V) &= (x -\dfrac{1}{D}) \dfrac{1}{D} V \end{aligned}\] Contoh:
\(\int x e^x \cos 2x dx=...\)

Rumus: Bukti:
Jika \(y = UV\) dimana \(U=f(x)\) dan \(v=g(x)\) maka turunan tingkat ke \(n\),
\(y^{(n)} = D^{(n)} (UV)\) dirumuskan oleh Leibnitz sbb: \(D^{(n)} = UD^nV + nDUD^{n-1}V+\dfrac{1}{D^2!} n(n-1)D^2UD^{n-2}V+\dfrac{1}{D^3!} n(n-1)(n-2)D^3UD^{n-3}V+..\)
Dengan memasang n=-1, maka:
\(\dfrac{1}{D}(UV)=U\dfrac{1}{D}V-DU\dfrac{1}{D2}V+D^2U\dfrac{1}{D^3}V-D^3U\dfrac{1}{D^4}V+...\)

Integrasi fungsi pecah rasional

Integrasi fungsi pecah rasional

  1. Jika \(N(x) = (ax + b)(cx + d)\) maka dibawa ke bentuk \(\dfrac{A}{ax+b} + \dfrac{B}{cx+d}\).

  2. Jika \(N(x) = (ax+b)^2(cx+d)\) maka dibawa ke bentuk \(\dfrac{A}{ax+b} + \dfrac{B}{ax+b}^2\).

  3. Jika \(N(x) = (ax^2+bx+c)\) dengan \(D<\) maka dibawa ke bentuk \(\dfrac{A}{cx+d} + \dfrac{Bx+C}{ax^2+b}\).

Contoh:

  1. \(I=\int \dfrac{x-1}{(x+1)(x^2+1)}dx\)

  2. \(I=\int \dfrac{3x+1}{x^2+2x+1}dx\)

  3. \(I=\int \dfrac{23-2x}{x^2+9x-5}dx\)

Integrasi fungsi trigonometri

Formula operasi fungsi trigonometri

  1. \(\sin^2x + \cos^2x =1\)

  2. \(\sin{2x} = 2 \sin{x} \cos {x}\)

  3. \(\cos{2x} = \cos^2x - \sin^2x\)

  4. \(\sin^2x =\dfrac{1}{2}(1-\cos{2x})\)

  5. \(\cos^2x =\dfrac1{1}{2}(1+\cos{2x})\)

  6. \(\sin(-x) = - \sin{x}; ~~ \cos{-x}= \cos{x}\)

  7. \(\sin {\alpha x} \cos {\beta x} = \dfrac{1}{2}\bigl\{\sin(\alpha+\beta)x+\sin(\alpha-\beta)x\bigr\}\)

  8. \(\cos {\alpha x} \sin {\beta x} = \dfrac{1}{2}\bigl\{\sin(\alpha+\beta)x-\sin(\alpha-\beta)x\bigr\}\)

  9. \(\cos {\alpha x} \cos {\beta x} = \dfrac{1}{2}\bigl\{\cos(\alpha+\beta)x+\cos(\alpha-\beta)x\bigr\}\)

  10. \(-\sin {\alpha x} \sin {\beta x} = \dfrac{1}{2}\bigl\{\cos(\alpha+\beta)x-\cos(\alpha-\beta)x\bigr\}\)
