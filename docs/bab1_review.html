1. Pembagian, Perpangkatan dan Akar

Pembagian

  • Jika \(b \neq 0\), maka \(\cfrac{0} {b} = 0\), sebab \(b.0 = 0\)

  • Jika \(a \neq 0\), maka \(\cfrac{a} {0}\) tidak didefinisikan.

    Sebab andaikan \(\cfrac{a}{0} = m\) maka \(a = 0.m\), ini tidak ada nilai \(m\) yang memenuhi.

  • Pernyataan \(\cfrac{0}{0}=TAK \, TENTU\), sebab andaikan \(\cfrac{0}{0} = n\) maka \(0=0.n\), Jadi nilai n tidak tunggal.

Perpangkatan

  • \(a^{m} . a^{n} =a^{m+n}\)

  • \((ab)^{n} = a^{n}+b^{n}\)

  • \((a^{m})^{n} = a^{mn}\)

  • Jika \(a \neq 0\) dan berhingga maka \(a^{0}=1\)

  • Jika \(a \neq 0\) maka \(\cfrac{a^m}{a^n}=a^{m-n}\)

  • Jika \(a \neq 0\) maka \(a^{-n}=\cfrac{1}{a^{n}}\)

Akar

  • Jika \(n\) bilangan bulat positif yang memenuhi \(a^n=b\), maka a disebut akar ke \(n\) dari \(b\). Ditulis \(a=\sqrt[n]{b}\) atau \(a=b^\frac{1}{n}\).

    1. \({\sqrt[n]{a}}^n = a\)

    2. \(\sqrt[n]{\frac{a}{b}}=\frac{\sqrt[n]{a}}{\sqrt[n]{b}}\)

    3. \(\sqrt[n]{ab}=\sqrt[n]{a} . \sqrt[n]{b}\)

    4. \(\sqrt[m]{\sqrt[n]{a}}=\sqrt[mn]{a}\)

2. Persamaan Kuadrat

Persamaan Kuadrat

  1. \(x_{1,2}=\cfrac{-b \pm \sqrt{b^2-4ac}}{2a}\) atau \(x_{1,2}=\cfrac{-b\pm\sqrt{D}}{2a}\)

    Dimana \(D = b^2 - 4ac\) (diskriminan)

  2. Jika \(D > 0\) \(\rightarrow\) PK mempunyai dua akar berlainan.

  3. Jika \(D = 0\) \(\rightarrow\) PK mempunyai akar real kembar.

  4. Jika \(D < 0\) \(\rightarrow\) PK tidak punya akar real.

  5. \(x_1 + x_2 = - \cfrac{b}{a}\); \(x_1.x_2=\cfrac{c}{a}\)

3. Fungsi Kuadrat

Fungsi Kuadrat

  1. Puncak: \(P(- \frac{b}{2a}, - \frac{D}{4a})\)

  2. Jika \(a > 0 \rightarrow\) grafik berupa parabola yang membuka ke atas \(y_{min} = - \frac{D} {4a}\)

  3. Jika \(a < 0 \rightarrow\) grafik berupa parabola yang membuka ke bawah \(y_{max} = - \frac{D}{4a}\)

  4. Jika \(D > 0 \rightarrow\) y memotong sumbu x di dua titik yang berlainan

  5. Jika \(D = 0 \rightarrow\) y menyinggung sumbu x

  6. Jika \(D < 0 \rightarrow\) y tidak memotong sumbu x

  7. Jika \(D \leqq 0 \rightarrow\) y tidak memotong sumbu x di dua titik

Fungsi Kuadrat (Cont’d)

  1. Fungsi Kuadrat disebut definit positif jika grafik seluruhnya berada di atas sb x; Syaratnya: (1) \(a > 0\)

    (2) \(D <0\)

  2. Fungsi Kuadrat disebut definit negatif jika grafik seluruhnya berada di bawah sb x; Syaratnya: (1) \(a < 0\)

    (2) \(D > 0\)

4. Logaritma

Logaritma Sifat-sifat:

  1. \(a^{^{a}\log{b}}=b\)

  2. \(^{a}\log{a}=1\)

  3. \(^{P}\log{a}+^{P}\log{b}=^{P}\log{ab}\)

  4. \(^{P}\log{a}-^{P}\log{b}=^{P}\log{\frac{a}{b}}\)

  5. \(^{P}\log{a^n} = n ^{a}\log{a}\)

  6. \(^{a}\log{a^n} = n\)

  7. \(^{a}\log{1} = 0\)

  1. \({^{a}\log{b}}=\frac{^{P}\log{a}}{^{P}\log{b}}\)

  2. \(^{a}\log{b}^{b}\log{c}=^{a}\log{c}\)

  3. \(^{a}\log{b}^{b}\log{c}=1\)

  4. \(^{a}\log{\sqrt[n]{b}=\frac{1}{n}^{a}\log{b}}\)

  5. \(^{a}\log{b} = \frac{1} {^{b}\log{a}}\)

  6. \(^{a}\log{\frac{1}{b}} = - ^{a}\log{b}\)

7. Satuan Imaginer dan Perkalian Istimewa

Satuan Imaginer

  • Bilangak kompleks: \(z=a+bi\); \(a=\) bagian real dari bilangan komplek z, \(b=\) bagian imaginer dari z.

  • Ingat bahwa: \(\sqrt{ab}=\sqrt{a}.\sqrt{b}\), sehingga: \(\sqrt{-3}=\sqrt{-1}.\sqrt{3}=i\sqrt{3}\)
    \(\sqrt{-9}=\sqrt{-1}.\sqrt{9}=i\sqrt{9}=i.3=3i\)

  • Akar-akar dari PK: \(x^2+x+1=0\) adalah: \(x_{1,2}=\cfrac{-1\pm\sqrt{1^2-4(1)}}{2}=\cfrac{-1\pm\sqrt{-3}}{2}=\cfrac{-1\pm\sqrt{-3}{i}}{2}\) \(x_{1}=\cfrac{-1}{2}+\cfrac{\sqrt{3}{i}}{2}\) \(x_{2}=\cfrac{-1}{2}-\cfrac{\sqrt{3}{i}}{2}\)

8. Geometri Analitik Datar

Geometri Analitik Dasar

Garis Lurus

  1. Jarak \(A(x_A, y_A)\) ke \(B(x_B,y_B)\) adalah \(AB=\sqrt{(x_B-x_A)^2+(y_B-y_A)^2}\)

  2. Persamaan explisit garis lurus \(y=mx+n\)
    (m= Koefisien arah/bilangan arah)

  3. Persamaan impisit gari lurus \(ax+by+c=0\) dengan bilangan arah \(m=-\frac{a}{b}\)

  4. Jarak dari \(A(x_A, y_A)\) ke garis \(ax+by+c=0\) adalah
    \(d=\Bigl\vert\cfrac{ax_{A}+by_{A}+c}{\sqrt{a^2+b^2}}\Bigr\vert\)

  5. Persamaan garis lurus melalui 2 titik \(A(x_A, y_A)\) ke \(B(x_B,y_B)\)
    \(\cfrac{y-y_A}{y_B-y_A}=\cfrac{x-x_A}{x_B-x_A}\)

Garis Lurus(Cont’d) Garis lurus \(g: ax+by+c=0\) dengan bilangan arah m1,
Garis lurus \(h: px+qy+r=0\) dengan bilangan arah m2;
maka supaya
\(g\) // \(h\), syaratnya: \(m_1=m_2\)
\(g\perp h\), syaratnya: \(m_1.m_2=-1\)
\(g\) memotong \(h\), syaratnya: \(m_1\neq m_2\)
\(g\) berimpit dengan \(h\), syaratnya: \(\cfrac{a}{p}=\cfrac{b}{q}=\cfrac{c}{r}\)

Lingkaran

  1. Persamaan lingkaran pusat \(0(0,0)\), jari-jarinya \(a\) adalah : \(x^2+y^2=a^2\)

  2. Persamaan lingkaran pusat \(P(a,b)\), jari-jarinya \(r\) adalah
    \((x-a)^2+(y-b)^2=r^2\)

  3. Lingkaran: \(x^2+y^2+Ax+By+C=0\) mempunyai
    Pusat di \(P(-\frac{1}{2} A, -\frac{1}{2} B)\); jari-jari \(r=\sqrt{\frac{1}{4} A^2 + \frac{1}{4} B^2 -C}\)

    Contoh Lingkaran: \(x^2+y^2-2x-4y+1=0\) mempunyai titik pusat
    di \(P(1,2)\) dengan jari-jari \(r=\sqrt{\frac{1}{4} {-2}^2 + \frac{1}{4} {-4}^2 -1}=2\)

Parabola Parabola: Tempat kedudukan titik-titik yang berjarak sama terhadap sebuah titik dan sebuah garis yang tertentu.
Titik-titik itu disebut Fokus; garis itu disebut Direktrix.

Ambil SR=sb x; SF=p dan OS = OF = \(\frac{1}{2} p\)
F\((\frac{1}{2}p,0)\) fokus; P(x,y) pada parabola.
Pada \(\Delta\) siku-siku PFR:
\(PF^2=PR^2+FR^2\)
\((x+\frac{1}{2}p)^2=y^2+(x-\frac{1}{2}p)^2\)
\(x^2+px+\frac{1}{4}p^2=y^2+x^2-px+\frac{1}{4}p^2\)

Parabola(Cont’d)


p = parameter parabola

Jika puncak parabola (a, b) dan sb. simetri tetap \(//\) sb. x, maka persamaan parabolanya:

\((y-b)^2 = 2p (x-a)\)

Ellips Ellips: Tempat kedudukan titik-titik yang jumlah jaraknya terhadap 2 titik tertentu tetap nilainya.

Fokus: F(-c,0) dan G(c,0)
P(x,y) pada ellips maka:
PF + PG = 2a (tetap)
Kedua titika A dan B memenuhi sebab
jika AF = GB = a - c, maka:
AF+AG=BF+BG
=(a-c)+(a+c)=2a

\(PF=\sqrt{(x+c)^2+y^2}\) dan \(PG=\sqrt{(x-c)^2+y^2}\)
\(PF+PG=2a \rightarrow (a^2-c^2)x^2+a^2+y^2=a^2(a^2-c^2)\)

Ellips(Cont’d) Misalkan: \(a^2-c^2=b^2\)
maka persamaan ellips: 2a = Sb. panjang
2b = Sb. pendek
Jika pusat ellips (\(\alpha,\beta\)) dan sumbu-sumbu simetri tetap \(//\) sb. x dan sb. y, maka persamaan ellipsnya:

\(\cfrac{(x-\alpha)^2}{a^2}+\cfrac{(y-\beta)^2}{b^2} = 1\)

Hyperbola Hyperbola ialah tempat kedudukan titik-titik yang selisih jaraknya terhadap dua titik tertentu tetap nilainya.

Fokus: \(F(-c,0)\) dan \(G(c,0)\)
\(AF=BG=c - a\)
\(AG-AF= BF - BG\)
\(= (c+a) - (c-a)\)
\(= 2a\)
P(x,y) pada hyperbola
\(PF = \sqrt{(x+y)^2 + y^2}\)
\(PG =\sqrt{x-c)^2+y^2}\)

Hyperbola(Cont’d) \(PF - PG=2a \rightarrow (c^2 - a^2)x^2-a^2-y^2=a^2(c^2-a^2)\)
Misalkan: \(c^2-a^2 = b^2\)
Maka persamaan hyperbola:
Jika pusat hyperbola (\(\alpha, \beta\)) dan sumbu-sumbu simetri tetap \(//\) sumbu x dan sumbu y; maka persamaan parabolanya:

\(\cfrac{(x-\alpha)^2}{a^2} - \cfrac{y^2}{b^2} = 1\)

Jika a = b; disebut hyperbola ORTHOGONAL (siku-siku).

9. Fungsi, Nilai mutlak bilangan real, notasi fakulteit dan radian

Fungsi

  • Definisi:
    Variabel y disebut fungsi dari variabel x jika dapat ditentukan hubungan anatara x dan y sedemikian hingga untuk setiap nilai x (yang mungkin diberikan) menentukan secara tunggal nilai y.


  • \(y \rightarrow\) variabel tak bebas

  • \(x \rightarrow\) variabel bebas

Nilai Mutlak dari bilangan real

  • Definisi:
    \(\left| x \right| = \begin{cases} x, \, jika \, x \geq 0 &\\ \\ -x, \, jika \, x < 0 \end{cases}\)

    Contoh: \(\vert 6 \vert = 6\); sebab \(6 >0\)
    \(\vert -5 \vert = -(-5)\); sebab \(5 \geq 0\)

Notasi Faktorial / Fakulteit Definisi:

Contoh:

  1. \(1! = 1\);

  2. \(3! = 3 . 2 .1=6\)

  3. \(0! = 1 (khusus)\)

Radian

  • Lingkaran satuan, jari-jari = 1

  • \(\cap \, AB = 1 \longrightarrow \angle AOB = 1\) radian

  • \(2 \pi rad = 360^o, \hskip10pt \frac{1}{3} \pi rad= 60^o\)

  • \(\pi rad = 180^o, \hskip15pt \frac{1}{4} \pi rad= 45^o\)

  • \(\frac{1}{2} \pi rad = 90 ^o, \hskip15pt \frac{1}{6} \pi rad= 30^o\)

Contoh: 1 radian = \(\cfrac{360}{2\pi} = \cfrac{360^o}{2(3.14159)}=57^o:\) 7’: 45"
